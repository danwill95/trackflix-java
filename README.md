## 📋 Sobre o Projeto

Trackflix é um sistema web completo para gerenciar sua biblioteca pessoal de filmes e séries. Esta versão é uma conversão do projeto original em Python/Flask para Java com Spring Boot, mantendo todas as funcionalidades e adicionando a robustez do ecossistema Spring.

### 🎯 Características

- ✅ **CRUD Completo**: Adicione, edite, visualize e remova filmes/séries
- ✅ **Busca Avançada**: Filtre por título, gênero, status ou avaliação
- ✅ **Sistema de Avaliação**: Avalie de 1 a 5 estrelas
- ✅ **Status Tracking**: Marque como "Assistindo", "Pendente" ou "Assistido"
- ✅ **Interface Moderna**: Design responsivo com Bootstrap 5
- ✅ **API RESTful**: Endpoints completos para integração
- ✅ **Persistência**: Armazenamento em arquivo JSON (pronto para migrar para banco de dados)
