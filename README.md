# 🎬 Trackflix - Gerenciador de Filmes Pessoal (Versão Java)

## 📋 Sobre o Projeto

Trackflix é um sistema web para gerenciar sua biblioteca pessoal de filmes e séries, desenvolvido em Java com Spring Boot.

## ✨ Funcionalidades

- ✅ Adicionar filmes/séries com detalhes completos
- ✅ Filtrar por status (Assistindo, Pendente, Assistido)
- ✅ Sistema de avaliação por estrelas (1-5)
- ✅ Busca por título, gênero ou notas
- ✅ Alterar status com um clique
- ✅ Interface responsiva e moderna
- ✅ API RESTful completa

## 🛠️ Tecnologias Utilizadas

- Java 17
- Spring Boot 3.1.5
- Thymeleaf
- Bootstrap 5
- jQuery
- Maven
